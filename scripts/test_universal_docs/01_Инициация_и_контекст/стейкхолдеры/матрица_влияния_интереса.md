## Динамика изменений стейкхолдеров

### Возможные перемещения между квадрантами

**Из "Информировать" в "Управлять вплотную":**
- Конечные пользователи могут получить больше влияния при сопротивлении изменениям
- Митигация: раннее вовлечение, демонстрация пользы

**Из "Поддерживать удовлетворенность" в "Управлять вплотную":**
- Финансовый директор может повысить интерес при превышении бюджета
- Митигация: прозрачная финансовая отчетность

**Из "Мониторить" в "Информировать":**
- Клиенты могут повысить интерес при изменении качества сервиса
- Митигация: мониторинг удовлетворенности

## Стратегии по квадрантам

### Коммуникационные каналы
| Квадрант | Частота | Формат | Содержание |
|----------|---------|--------|------------|
| Управлять вплотную | Еженедельно | Личные встречи, детальные отчеты | Прогресс, проблемы, решения |
| Поддерживать удовлетворенность | Ежемесячно | Краткие отчеты, презентации | Ключевые достижения, ROI |
| Информировать | Еженедельно | Email, демо, воркшопы | Новые функции, обучение |
| Мониторить | По необходимости | Опросы, общие уведомления | Общая информация |

### Управление рисками по квадрантам
- **Высокое влияние:** Риск блокировки проекта → Приоритетное внимание
- **Высокий интерес:** Риск чрезмерного вмешательства → Структурированное взаимодействие
- **Низкое влияние/интерес:** Риск неожиданного усиления → Регулярный мониторинг
